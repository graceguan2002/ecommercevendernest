# 电商服务商一体化工作台

为电商服务商提供一站式工作与信息获取平台。整体设计风格：白色底，蓝色突出重点。

## 包含功能

1. **通知板块**：重要通知同步，管理员可发布/编辑/删除
2. **规则板块**：5大分类内容管理
   - 引入规则（企业资质要求、报备流程）
   - 管理类内容（SAB、合伙人、外循环）
   - 风控信息（违规R值、Y值）
   - 工具类内容（二维码、百宝箱、最小作战单元、AI工具）
   - 服务商工单（加白、AI深度辅助申请说明）
3. **工具入口**：日常工具一键跳转
4. **服务商工单**：需求申请、查阅与处理

## 权限说明

- 登录管理员账号后可执行：
